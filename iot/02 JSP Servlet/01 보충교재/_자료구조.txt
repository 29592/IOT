자료구조  

    1.  Collection  : 하나씩 저장 
            1. Set     :   중복저장인 안되는 것  ex)  HashSet, TreeSet,
                               순서(x), 인덱스(x)  
            2. List     :   중복 저장이 됨.   순서(0) 인덱스(0) 
                              ex) ArrayList,    
                                    Vector,
                                    Stack :  Last  In  First  Out  

    2. Map :   키와 값 쌍으로 저장 . 
              ex)   HashMap,  TreeMap,  Hashtable,   Properties
